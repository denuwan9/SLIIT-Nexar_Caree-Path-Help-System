## 🚀 Key Features

### 👨‍💻 For Students
- **AI Career Advisor**: Personalized guidance powered by cutting-edge AI.
- **Dynamic Study Plans**: Auto-generated roadmaps tailored to your career goals.
- **Mock Interviews**: Real-time simulation and feedback for interview preparation.
- **Job Posting & Analysis**: intelligent job post tracking with AI-powered skill rating.
- **Interview Scheduling**: Seamless booking system for official campus interviews.

### 🛡️ For Administrators
- **Command Center**: A high-impact dashboard for system-wide oversight.
- **User Management**: Advanced registry for managing "Operators" (Students & Admins).
- **Student Analytics**: Deep-dive into student profiles, skill gaps, and engagement metrics.
- **Registry Reports 🆕**: Generate professional, fully detailed PDF reports of the user registry with a single click.
- **Audit Logs**: Comprehensive tracking of system activities and security protocols.

## 🏁 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- MongoDB Atlas account
- Gorq API Key 

### Installation

1. **Clone the Repository**
   ```bash
   git clone https://github.com/denuwan9/SLIIT-Nexar_Caree-Path-Help-System.git
   cd SLIIT-Nexar_Caree-Path-Help-System
   ```

2. **Setup Global Dependencies**
   ```bash
   npm run install-all
   ```

3. **Configure Environment Variables**
   - Create `.env` files in both `frontend/` and `backend/` directories.
   - Refer to `.env.example` in each folder for the required keys.

4. **Launch Development Environment**
   ```bash
   npm run dev
   ```
   *Frontend will be at `http://localhost:5173` and Backend at `http://localhost:5000`.*

## 🛡️ Security & Protocol

Nexar implements rigorous safety standards:
- **HttpOnly Cookies**: Secure token storage to prevent XSS.
- **Rate Limiting**: Protection against brute-force and DoS attacks.
- **Role-Based Access Control (RBAC)**: Fine-grained permissions for Students and Admins.
- **Data Hardening**: Input sanitization and encryption for all sensitive metadata.
